# 清流釣り ・ 河口釣魚（Three.js 3D 釣魚休閒遊戲）

面向一片風光明媚的河海合流，拿起釣竿靜待魚訊的日系小品。

## 開始遊玩

**最簡單**：直接用瀏覽器（Chrome / Edge）開啟 `index.html`（雙擊即可）。
three.js 由 CDN 載入，無需安裝、無需建置。

若瀏覽器擋下了 CDN 模組，改用本機伺服器：

```bash
npx serve .
# 或
python -m http.server 8000
```

然後瀏覽 `http://localhost:8000`。

## 操作

| 操作 | 動作 |
| --- | --- |
| **空白鍵 / 點擊畫面** | 拋竿；魚咬鉤時起竿 |
| **滑鼠移動** | 環顧四周（視角微幅平移） |
| **📖** | 開啟魚類圖鑑 |
| **🔊** | 開關音效 |

### 起竿時機
魚咬鉤時，浮標下沉、畫面出現一條**時機條**。
指標掃過時：
- 進入**綠色區段**起竿 → 釣獲成功
- 命中中央**黃心** → **完美**（價值 ×1.4）
- 太早 / 太遲 / 完全錯過 → 魚兒逃脫

## 特色

- **200+ 種魚**：40 個真實魚種家族 × 5 種變異（普通／斑紋／金線／深潭／白化），
  每種都有中日文名、稀有度、體長、重量、價值與詳細生態描述。
- **稀有度系統**：常見 → 普通 → 稀有 → 史詩 → 傳說，越稀有越難釣、價值越高。
- **圖鑑收集**：已釣獲的魚會解鎖，可隨時查看完整資料；進度自動存於瀏覽器。
- **日系乾淨畫風**：漸層天空、柔波水面著色器、圓潤的山丘與樹木、平滑的魚體模型，
  避免生硬菱角。

## 想自己調整？

所有內容都在 `index.html` 一個檔案裡，分段註解清楚：

- **魚類資料**：搜尋 `const FAMILIES`（每筆是一個魚種家族）、`const VARIANTS`（變異規則）。
  想新增魚種，照格式加一列即可，數量自動擴充。
- **難度 / 時機**：`const SWEEP`（時機條速度，毫秒）、`ZONE` / `PERFECT`（綠區、黃心範圍）。
- **咬鉤等待時間**：`startWait()` 內的 `biteAt`。
- **場景外觀**：搜尋 `天空`、`水面`、`河岸`、`遠山` 等中文註解。
- **釣竿位置**：`rod.position` / `rod.rotation`。
- **拋投落點**：`const CAST`。

> 提示：稀有度權重在 `const RARITY` 的 `weight` 欄位，數字越大越常出現。
